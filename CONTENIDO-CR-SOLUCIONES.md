## 7. Dónde usar cada bloque en el template

| Contenido | Uso sugerido en la web |
|-----------|-------------------------|
| §1 Identidad y contacto | Navbar, footer, hero, página de contacto |
| §2 Encabezado principal | Hero de la home, meta description |
| §3 Lista extrusión caucho | Sección “Productos” o “Servicios”, categoría Extrusión caucho |
| §4 Producto destacado | Banner o card destacado (marginadores molino caucho) |
| §5 Descripción de servicios | About / Nosotros, bloque “Qué hacemos” |
| §6 Lista resumida | Features, “Nuestros servicios”, footer |
| §6 Frase de cierre | Footer, bajo listados, CTA |
